# E-Commerce Application CURD Oprations.
RESTful APIs for CRUD operations on Product and Category entities with one-to-many relationships, using Spring Boot and JPA/Hibernate.
Integrated server-side pagination via Spring Data’s Pageable for efficient data retrieval and H2 RDBMS for persistent storage. 

# Endpoints for Categories: 
Getting All Categories details : http://localhost:8081/api/categories <br>
Getting categorie details by it's ID : http://localhost:8081/api/categories <br>
Updateting categories by it's ID : http://localhost:8081/api/categories/{id} <br>
Deleting category : http://localhost:8081/api/categories/{id} <br>

# Endpoints for Products: 
Getting All Products details : http://localhost:8081/api/products <br>
Getting Products details by it's ID : http://localhost:8081/api/products <br>
Updateting Products by it's ID : http://localhost:8081/api/products/{id} <br>
Deleting Product : http://localhost:8081/api/products/{id}
